Firebase Hosting 배포 방법
  1. Node.js 설치 후 터미널에서 npm install -g firebase-tools
  2. 한 번만 firebase login으로 Google 계정 연결
  3. 이 폴더(neo)에서 firebase deploy --only hosting 실행
    Firestore 보안 규칙도 같이 올리려면 firebase deploy --only hosting,firestore:rules
  4. 배포 전에 루트에 firebase-config.js가 있어야 합니다. 없으면 firebase-config.example.js를 복사해 채우세요.
  5. Firebase 콘솔 → Authentication → 설정 → 승인된 도메인에 프로젝트ID.web.app, 프로젝트ID.firebaseapp.com과 사용할 커스텀 도메인을 추가하세요.

Windows에서는 이 폴더의 deploy-hosting.ps1을 실행해도 됩니다.
